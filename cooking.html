🌊 挪威海鮮系列 (必吃)
1. 香煎檸檬奶油鮭魚
鮭魚經典10分
  1. 鮭魚擦乾，撒鹽和黑胡椒。
  2. 熱鍋融化奶油(Smør)，魚皮朝下放入。
  3. 中火煎 4 分鐘至皮酥脆，翻面再煎 2 分鐘。
  4. 起鍋前擠入檸檬汁，撒點香料。
2. 北極甜蝦開放三明治
甜蝦免煮5分
  1. 甜蝦(Reker)沖水解凍，自己剝殼 (這是樂趣)。
  2. 吐司稍微烤過，塗上一層厚厚的奶油或美乃滋。
  3. 鋪滿剝好的蝦子 (要堆得跟山一樣高)。
  4. 擠檸檬汁，配白酒，這是最道地的挪威吃法。
3. 奶香鮭魚燉飯 (一鍋到底)
鮭魚/生米濃郁25分
  1. 洋蔥碎炒香，加入生米拌炒。
  2. 分次加入水，邊煮邊攪拌約 15 分鐘。
  3. 米心微硬時，倒入半杯牛奶或鮮奶油(Fløte)。
  4. 鋪上切塊的鮭魚，蓋蓋子悶熟 5 分鐘。
4. 挪威魚湯 (Fiskesuppe)
魚湯包/鱈魚暖胃15分
  1. 超市買 "Lofoten Fiskesuppe" 湯包 (粉狀或液體)。
  2. 煮滾湯底，加入切塊的鱈魚、紅蘿蔔丁。
  3. 起鍋前加入一點鮮奶油和剝好的甜蝦。
  4. 配麵包沾著吃，極圈救命神湯。
5. 蒜香奶油煎鱈魚
鱈魚下酒10分
  1. 鱈魚肉質嫩易碎，不要太常翻動。
  2. 奶油爆香蒜末 (如有帶蒜頭)，放入鱈魚。
  3. 兩面煎金黃，淋上一點醬油提鮮。
  4. 非常適合配白飯。
🍖 肉食動物系列
6. 挪威肉丸佐馬鈴薯 (Kjøttkaker)
現成肉丸國民菜15分
  1. 超市買冷藏區的 "Kjøttkaker" (大肉丸)。
  2. 買一包 "Brun Saus" (棕色醬汁粉) 加水煮開。
  3. 放入肉丸燉煮 5-10 分鐘。
  4. 旁邊水煮馬鈴薯搭配吃，要有越橘醬(Tyttebær)更完美。
7. 可樂滷雞腿
雞腿台式30分
  1. 雞腿煎至表面金黃上色。
  2. 倒入可樂 (淹過雞肉一半) + 3 匙醬油。
  3. 大火煮滾，轉小火燉 20 分鐘。
  4. 最後開大火收汁，醬汁會變得很濃稠像糖醋。
8. 簡易肉醬義大利麵
牛絞肉西式15分
  1. 煮一鍋水煮義大利麵。
  2. 平底鍋炒香洋蔥和牛絞肉(Kjøttdeig)。
  3. 倒入一罐義大利麵醬 (Pasta Sauce)。
  4. 把煮好的麵拌進去即可。
9. 培根炒高麗菜
培根/高麗菜家常10分
  1. 熱鍋不放油，直接下培根片逼出油脂。
  2. 放入手剝的高麗菜 (挪威高麗菜較硬)。
  3. 加一點水蓋蓋子悶 2 分鐘。
  4. 加鹽或蠔油調味。
10. 週末塔可夜 (Taco Fredag)
牛絞肉/餅皮挪威傳統15分
  1. 週五去超市買 Taco Kit (包含餅皮、調味粉、莎莎醬)。
  2. 炒熟牛絞肉，加入調味粉和水。
  3. 切點生菜、番茄、小黃瓜、玉米粒。
  4. 自己包著吃，這是挪威人的週五儀式感。
🍜 亞洲胃救贖系列 (清冰箱)
11. 海鮮雜燴粥
剩飯/魚餅暖胃10分
  1. 水滾後加入康寶湯包 (從台灣帶)。
  2. 放入切塊的魚餅、冷凍蔬菜、剩飯。
  3. 煮到濃稠後，打個蛋花。
  4. 關火前放入甜蝦 (蝦煮久會縮)。
12. 沙茶肉燥乾麵
絞肉/麵條台式15分
  1. 洋蔥碎炒牛絞肉至焦香。
  2. 加入 2 匙沙茶醬、2 匙醬油、少許水。
  3. 小火燉 5 分鐘入味。
  4. 淋在煮好的麵條上，加一顆荷包蛋。
13. 黃金蛋炒飯
冷飯/蛋快速10分
  1. 蛋液打散，先下鍋炒半熟撈起。
  2. 爆香培根或火腿丁，放入冷飯炒散。
  3. 倒回雞蛋，加入鹽、黑胡椒。
  4. 如果有蔥花或生菜絲，起鍋前加入。
14. 醬燒魚排蓋飯
任何魚排日式15分
  1. 洋蔥切絲，鋪在平底鍋底。
  2. 放上魚排，淋上醬汁 (醬油2:糖1:水3)。
  3. 蓋蓋子中小火「蒸煮」8 分鐘。
  4. 連湯汁一起淋在白飯上，超下飯。
15. 簡易部隊鍋
泡麵/熱狗韓式10分
  1. 用辛拉麵當底。
  2. 加入切片熱狗 (Pølse)、魚餅、起司片、蔬菜。
  3. 全部煮成一鍋，冷天吃超爽。
🍳 早餐與輕食系列
16. 挪威熱狗捲 (Pølse i Lompe)
熱狗/軟餅經典5分
  1. 熱狗用水煮熱 (挪威人通常水煮)。
  2. 拿一張 Lompe (馬鈴薯軟餅)。
  3. 捲起熱狗，擠上番茄醬和芥末醬。
  4. 一定要灑上炸洋蔥酥 (Sprøstekt løk)！
17. 培根烘蛋 (Omelet)
蛋/培根早餐10分
  1. 培根煎脆，加入蔬菜丁炒軟。
  2. 倒入打散的蛋液 (可加一點牛奶更嫩)。
  3. 小火慢煎至底部凝固，對折起鍋。
18. 馬鈴薯肉末濃湯
馬鈴薯/絞肉西式20分
  1. 洋蔥和絞肉炒香，加入馬鈴薯小丁。
  2. 加水煮到馬鈴薯化開 (用湯匙壓泥增加濃稠度)。
  3. 加入牛奶煮滾，鹽巴黑胡椒調味。
  4. 沾麵包吃就是一餐。
19. 烤箱時蔬雞腿 (懶人版)
雞腿/根莖類烤箱40分
  1. 如果住宿有烤箱。
  2. 烤盤鋪上馬鈴薯塊、紅蘿蔔塊、洋蔥塊。
  3. 放上雞腿，全部淋油、撒鹽和香料。
  4. 200度烤 40 分鐘，中間翻面一次。完全不用顧火。
20. 鮪魚玉米蛋沙拉
鮪魚罐頭便當10分
  1. 水煮蛋切碎。
  2. 拌入鮪魚罐頭、玉米粒、美乃滋、黑胡椒。
  3. 夾在土司裡，是明天開車路上最好的午餐。
